## -AGENDO PROJECT-

### Descripción
AGENDO es un sistema de reservas y gestión de citas para negocios que permite programar, modificar y notificar citas a clientes y usuarios.

### Objetivo del Proyecto
Desarrollar una aplicación web que facilite la gestión eficiente de citas y servicios ofreciendo recordatorios y reportes.

### Tecnologías
- Frontend: React JS
- Backend: Laravel, PHP
- Base de Datos: MySQL
- APIs externas: Twilio (SMS), Google Calendar

### Metodologías
- Scrum para gestión del proyecto
- Kanban para flujo visual de actividades

### Alcance
El sistema incluirá módulos para:
- Gestión de citas
- Administración de servicios y usuarios
- Notificaciones automáticas vía SMS y correo
- Generación de reportes
